# 🧺 Laundry Renovation Hub

Planning and tracking note for the laundry room renovation.

---

## 🎯 Goals

- Improve storage and workflow in the laundry room
- Modernise layout for better daily usability
- Stay close to existing plumbing to minimise cost

---

## 📐 Planning Tasks

### Measurements & Layout

- [ ] Measure room dimensions (width, depth, height) [priority:: 1]
- [ ] Measure existing appliances — washing machine, dryer [priority:: 1]
- [ ] Photograph current layout and plumbing locations [priority:: 1]
- [ ] Sketch layout options — appliance placement vs storage vs bench [priority:: 2]

### Plumbing & Electrical

- [ ] Confirm existing tap and waste outlet locations [priority:: 1]
- [ ] Decide whether to move plumbing or design around it [priority:: 2]
- [ ] Count power points needed — washing machine, dryer, iron, extras [priority:: 2]
- [ ] Check if additional power points or a new circuit is required [priority:: 2]

### Storage & Cabinetry

- [ ] Define what needs to be stored — brooms, vacuum, ironing board, cleaning supplies [priority:: 2]
- [ ] Decide on cabinetry style — flat-pack vs custom vs freestanding [priority:: 3]
- [ ] Get quotes from Bunnings, IKEA, or local joiner [priority:: 3]

### Fixtures & Finishes

- [ ] Choose laundry tub/sink — drop-in vs undermount, size [priority:: 3]
- [ ] Choose splashback material [priority:: 3]
- [ ] Choose flooring — tile, vinyl, or keep existing [priority:: 3]
- [ ] Consider hanging rail or rod for drying [priority:: 3]

### Ventilation & Lighting

- [ ] Check existing ventilation — exhaust fan present? [priority:: 2]
- [ ] Assess natural light — window present? [priority:: 2]
- [ ] Plan lighting — overhead + task lighting [priority:: 3]

## 📝 Notes & Research

- Keeping plumbing in existing location saves significant cost — design around it first
- One power point per appliance minimum; plan for extras to avoid retrofitting
- Vertical storage maximises a small space
